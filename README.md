
# Calculator Web Application

Designed and developed an interactive calculator that enables users to perform mathematical calculations efficiently. The project demonstrates JavaScript event handling, DOM manipulation, responsive web design, and dynamic expression evaluation using built-in JavaScript functions.


## Features

- Basic arithmetic operations (+, −, ×, ÷)
--Percentage calculation
- AC (Clear All)
- DEL (Delete Last Character)
- Real-time display updates
- Responsive and modern interface
- Hover animations for buttons
- Dynamic expression evaluation using JavaScript


## Tech Stack

**Frontend:** HTML5, CSS3, JavaScript (ES6)

**Tools:** Visual Studio Code
